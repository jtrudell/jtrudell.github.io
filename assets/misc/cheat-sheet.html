Objects, Functions, and Loops

How to create objects in Javascript

Scenario: make a My Little Pony object
Object Literals

    var twilightSparkle = {
      name: "Twilight Sparkle",
      typeOfPony: "alicorn",
      color: "purple",
      hasCutieMark: true
    }
Object Prototypes

Scenario: create a template to reuse the pony object...because of course we want to easily be able to make lots of ponies!


    function pony(name, type, color, mark) {
      this.name = name;
      this.type = typeOfPony;
      this.color = color;
      this.mark = hasCutieMark
    }

    var rarity = new pony("Rarity", "unicorn",
                  "silvery white", "true");

How to define a function in Javascript

Scenario: create a function that will make a pony object say hi to you
Simple function

    function sayHi(aPony) {
      console.log(aPony.name + " says hi!");
    }

    sayHi(twlightSparkle);
    // "Twilight Spark says hi!"

How to loop in Javascript

Scenario: add an exclamation mark to each pony name in an array
for loops

    var names = ["Twilight", "Rarity", "Rainbow Dash"];

    for (var i = 0; i < names.length; i++) {
      names[i] += "!";
    }
while loops

    var i = 0;

    while (i < names.length) {
      names[i] += "!";
    }
for in loops

Note: do not use this if the order of the elements in the array matters, because this type of loop will iterate over elements in an arbitrary manner.


    for (var i in names) {
      names[i] += "!";
    }
do while loops

Note: will always execute the first statement once, even if the while condition isn't true; if the condition is true, will continue to execute the statement until it is not true.


    do {
      console.log("Ponies are awesome!")
      }
      while (true);
    // creates an infinite loop. If condition
    // were false, "Ponies are awesome!" would
    // still execute one time.
